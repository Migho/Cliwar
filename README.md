# Cliwar

Tämä projekti on Helsingin yliopiston erään opiskelijan ohjelmoinnin harjoitustyö. Työn tavoitteena on tehdä graafinen nopeuspeli, jossa kaksi pelaajaa nostelevat kortteja vaikuttaakseen pelin kulkuun.
